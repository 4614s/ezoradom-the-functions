EzoraDOM Dökümantasyonu 4: Element oluşturma Bölüm 1

EzoraDOM'da element oluşturmak için öncelikle şu iki şeyi bilmelisiniz.


  1. EzoraDOM'da bir parametreye 0 değeri vermek. Ona değer vermemekle aynıdır (Bazı durumlar hariç). "" vb. değerler yerine 0 girmek daha çok tercih edilir. 0 değeri number olmalıdır.
  2. EzoraDOM'da yeni oluşturulan elemente id ve class ismi vermek zorunludur. Aksi taktirde programı çalıştıramazsınız

Element oluşturma:

ez.createElement(Tag ismi, İçeriği, Id, Class ismi, Konumu);

1. Parametre: Oluşturulacak elementin etiket ismidir. String olmalıdır.

2. Parametre: Oluşturulam elementin içeriğidir. String olmalıdır.

Id Parametresi: Oluşturulan elementin idsidir. Verilmek istenmezse number tipinde 0 girilir. Verilmez istenirsede string tipinde değer girilir. Her fonksiyonda id parametresi aynıdır.


Class ismi parametresi: Oluşturulan elementin class ismidir. Verilmek istenmezse number tipinde 0 girilir. Verilmez istenirsede string tipinde değer girilir. Her fonksiyonda class ismi parametresi aynıdır.


Konum parametresi: Oluşturulan elementin konumudur. Değer girilmezse bodyye yerleştirilir. Değer girilirse query selector olarak string değer girilir. Her fonksiyonda konum parametresi aynıdır.


Hyperlink (a elementi) oluşturma

ez.createLinkElement(Yazısı, Href değeri, Id, Class ismi, Konum);

1. Parametre: Elementin yazısıdır.

2. Parametre: Href yani tıklanınca yönlendirilecek değerdir. 0 kuralı geçmeyen nadir parametrelerden biridir.



Buton oluşturma

ez.createButton(Yazısı, Onclick değeri, Id, Class ismi, Konum);

1. Parametre: Buton yazısıdır.

2. Parametre: Onclick değeridir. Tıklandığında gerçekleşecek fonksiyonun değeridir. direkt "fonksiyonIsmi()" gibi fonksiyonlarda girebilirsiniz string olmalıdır. 0 kuralı geçerlidir.



Sayfaya fotoğraf ekleme

ez.createImage(Src (kaynak), Alt bilgisi, Id, Class ismi, Konum);

1. Parametre: Src yani kaynak parametresi. Girilmesi zorunludur.

2. Parametre: Alt bilgi parametresi ekran okuyucular için önemlidir. Tabi 0 girerek direkt değer verilmeyedebilir.
