EzoraDOM dökümantasyonu 12- Semantik element oluşturma (Element oluşturma bölüm 3)

Semantik elementler çok önemlidir. Kodun okunabilirliğini arttırır ve sayfaya anlam katar. EzoraDOM'da da vardır. v1.1 sürümü ile geldiği için v1 sürümünde bulunmamaktadır.


Paragraf:

ez.createParagraph(İçerik, Id, Class ismi, Konum);

Hyperlink:

ez.createHyperLink(Yazı içeriği, Href, Id, Class ismi, Konum);
// Aslında bu createLinkElement() ile aynı işlevde ama bunu öneririz.

Section & Article (Aside yok)

ez.createSection(İçerik, Id, Class ismi, Konum);

ez.createArticle(İçerik, Id, Class ismi, Konum);

Header & Footer (Main yok)

ez.createHeader(İçerik, Id, Class ismi);

ez.createFooter(İçerik, Id, Class ismi);

// Bunlarda konum yok çünkü bir header elementinin sayfanın sonuna yerleştirilmesi garip olurdu.

Farkettiyseniz bu sayfada semantik etiketlerin anlamlarını anlatmadık. Bu sayfayı en verimli şekilde anlamak için önce Element oluşturma 1. bölümü daha önceden okumuş olmanız ve HTML semantik etiketlerini bilmeniz gerekmektedir.
