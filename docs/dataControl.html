EzoraDOM dökümantasyonu 10: Veri depolama

Bu konuda yerel depolama alanı kullanılır. Session storage kullanılmaz ve sadece 3 tane fonksiyonumuz var.

// Veri ayarlamak (Yoksa atar varsa değiştirir)
ez.setData(Key, Value);

// Veri okuma (değer döndürür)
ez.getData(Key);

// Veri silme
ez.removeData(Key);

// Verileri temizleme. v1.1 sürümü ile geldi.
ez.clearDatas();
