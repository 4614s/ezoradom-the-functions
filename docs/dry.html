EzoraDOM dökümantasyonu 11: Tekrar fonksiyonu

EzoraDOM'da kopyala yapıştır yapmana gerek yok. For döngüsü ile uğraşmanızada gerek yok. Kullanabilirsinizde ama daha kolayı var

repeat fonksiyonu

ez.repeat(Tekrar sayısı, Fonksiyon);

Örnek: 15 tane element oluşturma

ez.repeat(15, () =>{
ez.createElement("p", `Hello world ${i}!`, 0, 0);
let i = 1;
i++;
});

Ek bilgi: print ile printBl fonksiyonlarının 2. parametresi tekrar parametresidir.

ez.print("Hello!", 5);

ez.printBl("Hello!", 5);
