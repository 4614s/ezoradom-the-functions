EzoraDOM Dökümantasyonu 9: Events (Olaylar)

Event ekleme

// Idye göre event ekleme
ez.addEventById(Id, Event, Fonksiyon);

// Class ismine göre event ekleme
ez.addEventByClassName(Class ismi, Event, Fonksiyon, Seçim parametresi);

// Tag ismine göre event ekleme
ez.addEventByTagName(Tag ismi, Event, Fonksiyon, Seçim parametresi);

// Query Selectore göre event ekleme
ez.addEventByQuerySelector(Target, Event, Fonksiyon, Seçim parametresi);

Örnek: Tüm butonlara tıklanınca uyarı "Butona tıkladın." mesajı çıkacak

ez.addEventByTagName("button", "click", () => {
alert("Butona tıkladın.");
}, "all);

Event silme

// Idye göre event silme
ez.removeEventById(Id, Event, Fonksiyon ismi);

// Class ismine göre event silme
ez.removeEventByClassName(Class ismi, Event, Fonksiyon ismi, Seçim parametresi);

// Tag ismine göre event silme
ez.removeEventByTagName(Tag ismi, Event, Fonksiyon ismi, Seçim parametresi);

// Query Selectore göre event silme
ez.removeEventByQuerySelector(Target, Event, Fonksiyon ismi, Seçim parametresi);

Uyarı: Lütfen şu hatayı yapmayın

// Hatalı
ez.addEventByTagName("button", "click", () => {
alert("Butona tıkladın.");
}, "all);

ez.removeEventByTagName("button", "click", () => {
alert("Butona tıkladın.");
}, "all");

Yukardaki kod referans hatası verir. Çünkü eklenen kod ile silinecek kod boşlukları dahil aynı bile olsa farklı referanslara sahiptir. Aşağıdaki kodu kullanın

function showAlert() {
alert("Butona tıkladın.");
}

ez.addEventById("btn", "click", showAlert);

ez.removeEventById("btn", "click", showAlert);

Yukarısı hatasızdır. Çünkü fonksiyon tanımlanır. Sonra tıklama eventi eklenir. Sonra aynı şekilde silinir. Fonksiyon tek kez tanımlandığı için aynı fonksiyon eventa eklenip sonradan aynı fonksiyonda silinir.
