EzoraDOM Dökümantasyonu 7: Head içeriği

Bilgilendirme: ez.createElement() fonksiyonunda genellikle konum parametresine "body" yazılmaz çünkü o parametre boş olsada aynısı olur ve genellikle head ici element oluşturmak için kullanılmaz.

Şimdi EzoraDOM'da head içeriği nasıl ayarlanır bakalım.


Sayfa dili belirleme (Seo açısından önemlidir)

ez.pageLang(Sayfa dili); // "tr" yazmak Türkçe yapar mesela.

Sayfa başlığı belirleme

ez.pageTitle(Sayfa başlığı);

Sayfa karakter seti (charset) belirleme

ez.pageCharset(Sayfa charseti);

Sayfa faviconu belirleme

ez.pageFavicon(Src);

Meta element oluşturma

ez.createMetaElement(Name / Http-equiv, Content);
/*
İlk parametreye "refresh" veya "X-UA-Compatible" ise http-equiv değilse name attribute değeri olur.

Content değişmez
*/
