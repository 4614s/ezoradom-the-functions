EzoraDOM Dökümantasyonu - Asetler: Canlı saat

ez.createLiveClock(Font-size, Id, Class name, Konum);

1. Parametre oluşturulacak canlı saatin font boyutunu belirtir.

Girilen değer number tipinde ise girilen değer×10px font verilir 2.5 girilirse 25px olur.

Eğer değer string ise özel değer girilir hatta clamp() gibi css fonksiyonları bile girilebilir.


Diğer Id, Class ismi ve Konum parametreleri element oluşturma bölümündeki ile aynıdır.



Örnek kod

ez.createLiveClock("3rem", 0, 0, "#alan");
