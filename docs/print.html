EzoraDOM Dökümantasyonu: 1- ez.print() ile ez.printBl()

EzoraDOM'u sayfaya dahil etmeyi biliyorsanız seçtiğiniz CDN scriptini head etiketinin içine ekleyin. Korkmayın hata vermez.

Dahil etmeyi bilmiyorsanızda Bundan bir önceki dökümana gidin. Ordan öğrenebilirsiniz.



Dökümana (HTML sayfasına) birşey yazdırma:

Dökümanınıza (HTML sayfanıza) birşey yazdırmak istiyorsanız aşağıdaki kodlar en basit olanlarıdır.

ez.print(Yazdırılacak içerik);
ez.printBl(Yazdırılacak içerik);

ez.print() içeriği inline olarak yazdırırken.
ez.printBl() içeriği block olarak yazdırır.

Yani

ez.print("Hello world!");
ez.print("Hello world!");
ez.print("Hello world!");

Üstteki kod yan yana 3 tane Hello world! yazdırırken.

ez.printBl("Hello world!");
ez.printBl("Hello world!");
ez.printBl("Hello world!");

İse alt alta 3 tane Hello world! yazdırır.


Denemesi:

Çıktı (ez.print(); ile )
Hello world!Hello world!Hello world!

Çıktı (ez.printBl(); ile )
Hello world!
Hello world!
Hello world!

EzoraDOM'da tepmlate literals çok kullanılır.

//Örnek
let name = prompt("İsminizi giriniz:");
ez.print("Hoşgeldin",name,"!");
/*Yukarısı hatalıdır undefined döner.
Çünkü Javascript name ile "!" işaretini parametre olarak alır. Yani yazınıza dahil etmez ve yazdırmaz. */

//Ya + operatörü kullanılır.
ez.print("Hoşgeldin" + name + "!");

//Ya da tepmlate literals
ez.print(`Hoşgeldin ${name} ! `);
