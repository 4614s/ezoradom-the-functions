EzoraDOM dökümantasyonu 6: Element silme

Aslında çok basit

// Idye göre silme
ez.removeElementById(Id);

// Class ismine göre silme
ez.removeElementByClassName(Class ismi, Seçim parametresi);

// Etiket(tag) ismine göre silme
ez.removeElementByTagName(Tag ismi, Seçim parametresi);

Query Selector ile silme
ez.removeElementByQuerySelector(Target, Seçim parametresi);
