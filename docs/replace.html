EzoraDOM Dökümantasyonu 3: Replace Fonksiyonları

DOM'un en ilk öğrenilenleri innerHTML ve innerText(textContentte aynı sayılır). EzoraDOM'daki karşılığı nedir?

replaceText ile replaceHTML. Bu iki ana fonksiyonun seçiciler ile beraber toplamda tane 8 replace fonksiyonu vardır.



replaceText ne yapar?

Seçtiği elementin yazısını değiştirir .

// Idye göre değiştirme
ez.replaceTextById(id, Yeni yazı);

// Class ismine göre değiştirme
ez.replaceTextByClassName(Class ismi, Yeni yazı, Seçim parametresi);

// Tag(etiket) ismine göre değiştirme
ez.replaceTextByTagName(Tag (etiket) ismi, Yeni yazı, Seçim parametresi);

//QuerySelector ile değiştirme
ez.replaceTextByQuerySelector(Target (Hedef), Yeni yazı, Seçim parametresi);



replaceHTML ne yapar?

replaceTextten Tek farkı yazı içeriğini değil direkt HTML kodunu değiştirebilmesidir.

// Idye göre değiştirme
ez.replaceHTMLById(id, Yeni içerik);

// Class ismine göre değiştirme
ez.replaceHTMLByClassName(Class ismi, Yeni içerik, Seçim parametresi);

// Tag(etiket) ismine göre değiştirme
ez.replaceHTMLByTagName(Tag (etiket) ismi, Yeni içerik, Seçim parametresi);

//QuerySelector ile değiştirme
ez.replaceHTMLByQuerySelector(Target (Hedef), Yeni içerik, Seçim parametresi);
