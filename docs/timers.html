EzoraDOM Dökümantasyonu - Asetler: Sayım Sayaçları

Geri sayım sayacı

Sayım bitince girdiğiniz fonksiyonu çalıştırır ve sayımı görmeniz için element oluşturur.

ez.createCountdownTimer(Süre (saniye), Fonksiyon, Element Id, Element Class ismi, Element Konumu);

İlk parametre belirlenen süredir. O süre bitince ikince parametredeki fonksiyon çalışır.

3, 4, 5. parametre oluşturulan sayım elementine aittir. Unutmayın element oluşturulduğu için id ile classname boş kalamaz.


İleri sayım sayacı

ez.createCountUpTimer(Süre (saniye), Fonksiyon, Element Id, Element Class ismi, Element Konumu);

Geri sayım sayacı ile neredeyse aynıdır. Ama tek farkı geriye değil ileriye sayar.



Örnek kod:

ez.createCountdownTimer(5, () => {
alert("5 saniye geçti");
}, 0, 0, "#alan");
